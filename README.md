# 在线考试后台管理系统 java715

## 项目概述

在线考试后台管理系统是基于Springboot与Vue开发的，适用于实现教育机构在线考试需求的一套系统。它集成了多种功能，能够满足管理员、老师与学生三个不同角色的操作需求。

## 技术栈

### 推荐环境配置
- IDEA
- JDK 1.8
- Maven
- MySQL

### 前端技术
- Vue
- Ajax
- JSON

### 后端技术
- SpringBoot
- MyBatis Plus

## 功能模块

### 系统角色
- 管理员
  - 拥有系统最高权限。
- 老师
  - 考试管理
  - 题库管理
  - 成绩管理
  - 学生管理
- 学生
  - 参与考试
  - 查看成绩
  - 试题练习
  - 留言

## 运行环境

- 服务器：推荐使用符合Java Web应用标准的服务器，如Tomcat。
- 数据库：MySQL版本需与系统兼容。
- IDE：根据个人习惯，推荐使用IDEA。

## 目录结构

```plaintext
├── backend                # 后端代码
│   ├── src
│   │   ├── main
│   │   │   ├── java       # Java源代码
│   │   │   └── resources  # 配置文件等资源
│   └── pom.xml            # Maven项目文件
└── frontend               # 前端代码
    ├── src
    │   ├── assets         # 资源文件
    │   ├── components     # Vue组件
    │   └── views          # 页面视图
    └── package.json        # Node.js项目文件
```
